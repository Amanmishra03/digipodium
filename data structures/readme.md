## list
  - mutable
  - indexed
  - ordered
  - can contain duplicate value
## dict
   - mutable
   - unindexed
   - ordered
   - can contain only unique keys
   - can contain duplicate value

   - mutable
   - unindexed
   - unordered 
   - can contain only unique value
# tuple
   - immutable 
   - indexed
   - ordered
   - can contain duplicates
